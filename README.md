A video game designed by Tashi Jampa. Made in JavaScript on JDK7.0_80 complier.


// Project Description && How to Play //
The game is a recreation of a well known mobile game called brick breaker, there are 	many versions of the game but I have recreated a simple version of the game. The objective of the game is to bounce a ball off of a landing pad and destroy blocks at the top of the screen, if the user were to miss the ball when it falls back down then they will lose a life, they will have three lives and once all the lives are gone they can restart. Also there is a score counter. When you destroy all of the blocks the blocks then regenerate creating an infinite game, enjoyable for forever. 

To play the user will need to move their arrow keys ( left and right ) to receive the ball and shoot it back up again. 
They will also need to download all of the image packs, blocks, ball etc.

Have fun!
